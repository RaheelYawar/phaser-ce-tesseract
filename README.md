# OCR using Tesseract.js with Phaser-CE 

## Install

### Clone this git repo
`git clone https://github.com/GamedevMania/phaser-ce-boilerplate.git`

### Install all dependencies
`npm install`

### Start dev server and develop your game!
`npm start`

### Build your game
`npm run build`
